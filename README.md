# College Study Hub - MERN Starter

This archive contains a MERN stack starter with OpenAI PDF summarization + TTS prototype.

Folders:
- backend: Node.js + Express + OpenAI + pdf-parse + google-tts-api
- frontend: React (Create React App)

Follow README files in each folder to run locally.

Important:
- Add your OpenAI key to backend/.env
- Start MongoDB locally or use Atlas and set MONGODB_URI
